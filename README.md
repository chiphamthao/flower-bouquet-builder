## Requirements
Flask

Python 3.6+

### Setup
1. Clone this repository.
2. Install Flask:
   ```bash
   pip install Flask
   ```
3. Run from the project root directory.
   
## Usage
To run the app, execute the following in the terminal:

```
python3 server.py
```
